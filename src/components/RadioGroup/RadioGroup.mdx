## Використання

Компонент потребує обгортки `<Formik>` — він читає/пише значення через `useField(name)`.

<Source language="tsx" code={`import { Formik, Form } from 'formik';
import RadioGroup from '@/components/RadioGroup';

const sizeOptions = [
  { value: 'sm', label: 'Малий' },
  { value: 'md', label: 'Середній' },
  { value: 'lg', label: 'Великий' },
];

<Formik initialValues={{ size: '' }} onSubmit={...}>
  <Form>
    <RadioGroup name="size" label="Розмір" options={sizeOptions} />
  </Form>
</Formik>`} />
