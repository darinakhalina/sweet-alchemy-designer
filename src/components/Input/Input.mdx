## Використання

Компонент потребує обгортки `<Formik>` — він читає/пише значення через `useField(name)`.

<Source language="tsx" code={`import { Formik, Form } from 'formik';
import Input from '@/components/Input';

<Formik initialValues={{ email: '' }} onSubmit={...}>
  <Form>
    <Input name="email" type="email" label="Email" placeholder="baker@gmail.com" />
  </Form>
</Formik>`} />
