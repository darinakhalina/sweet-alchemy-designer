## Використання

Компонент потребує обгортки `<Formik>` — він читає/пише значення через `useField(name)`.

<Source language="tsx" code={`import { Formik, Form } from 'formik';
import Switch from '@/components/Switch';

<Formik initialValues={{ darkMode: false }} onSubmit={...}>
  <Form>
    <Switch name="darkMode" label="Темна тема" />
  </Form>
</Formik>`} />
